# CEP_Lab_last

## Project Overview
A C/C++ laboratory project. Currently contains a minimal starter setup with a `main.c` entry point and a `Makefile` for building.

## Build System
- **Language:** C (GCC 14.3.0)
- **Build tool:** Make
- **Compiler flags:** `-Wall -Wextra -std=c11`

## Project Structure
```
.
├── main.c       # Main C source file (entry point)
├── Makefile     # Build configuration
├── .gitignore   # C/C++ build artifact ignore rules
├── LICENSE      # MIT License
└── README.md    # Project title
```

## Running the Project
- Build: `make`
- Run: `make run`
- Clean: `make clean`

## Workflow
- **Start application**: Runs `make run` (console output)

## Notes
- Nix channel: `stable-25_05`
- No frontend; console-only application
